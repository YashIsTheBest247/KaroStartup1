# KaroPitch 🚀

KaroPitch is a modern startup–investor discovery platform designed to connect promising startups with investors looking for the next big opportunity.
This project contains the landing page for the KaroPitch platform built with **React, Vite, and Tailwind CSS**.

---

## 🌟 Features

* Modern responsive landing page
* Smooth scrolling navigation
* Sections for **About, How It Works, Startups, and Investors**
* Clean UI inspired by modern startup platforms
* Built with fast and lightweight technologies

---

## 🛠️ Tech Stack

* **React**
* **Vite**
* **Tailwind CSS**
* **TypeScript**

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/YashIsTheBest247/KaroStartup1.git
```

Go into the project directory:

```
cd karo-pitch
```

Install dependencies:

```
npm install
```

---

## ▶️ Run the Development Server

```
npm run dev
```

The app will run on:

```
http://localhost:8080
```
